# 💥 最強Todo

タスクを叩きつけて、完了で爆発させて、削除で雷を降らす最強のTodoアプリ。

## 🔗 デモ

**👉 [https://Shigeta012103.github.io/saikyo_todo_260318/](https://Shigeta012103.github.io/saikyo_todo_260318/)**

## 操作方法

- **追加**: テキスト入力 → Enter（叩きつけ + 画面揺れ）
- **完了**: チェックボックスをクリック（爆発エフェクト）
- **削除**: × ボタン（雷エフェクト）

## 技術スタック

- React + TypeScript + Vite
- Framer Motion（リストアニメーション）
- Canvas API（爆発・雷エフェクト）
